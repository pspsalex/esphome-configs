# ESPHome Configs

This repository contains ESPHome configurations for various devices:
- M5Paper
- CYD (Cheap Yellow Display)
- ~~M5Dial (Knob)~~ (TODO)
- M5Paper SDL (simulates M5Paper display / layout)
